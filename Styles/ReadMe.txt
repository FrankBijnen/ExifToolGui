These styles try to mimic the original ExifToolGui 3 color options.

Install them to your dev environment by copying the vsf files to your installation styles directory.
E.G. C:\Users\Public\Documents\Embarcadero\Studio\22.0\Styles

Delphi 12 Athens notes:
- Some styles, like the MetropolisUI*.vsf are not distributed with Delphi. You can find them here.
  https://github.com/RRUZ/vcl-styles-utils/tree/master/Demos/Styles
- The style directory has an updated version number.
   E.G. C:\Users\Public\Documents\Embarcadero\Studio\23.0\Styles
